## 1.给变量赋值
### 1.1 赋值固定字符串 
```
F:\>set var=good

F:\>echo %var%
good

F:\
```
### 1.2 赋值用户输入字符
```
F:\>set /p var=
bad

F:\>echo %var%
bad

F:\>
```
### 1.3 赋值文本内容
```
F:\>echo badly>a.txt

F:\>set /p var=<a.txt

F:\>echo %var%
badly

F:\>
```
### 1.4 同时赋值多个变量(仅数字)
```
F:\>set /a a=b=c=d=55
55
F:\>echo %a%-%b%-%c%-%d%
55-55-55-55

F:\>
```
## 2.变量计算
### 2.1 加/减/乘/除/余
使用`set`指令进行计算时，需要添加`/a`参数，且只能对整数进行运算

给变量`a`和`b`赋值
```
F:\>set a=1 & set b=2
```
1.加法：
```
F:\>set /a c=%a%+%b%
3
```
2.减法：
```
F:\>set /a d=%a%-2
-1
```
3.乘法：
```
F:\>set /a e=%a%*%b%*3
6
```
4.除法：(结果==向下取整==)
```
F:\>set /a f=1/2
0
F:\>set /a g=5/2
2
```
5.取余:
```
F:\>set /a h=123%10
3
F:\>
```
